Analyzing Historical Stock/Revenue Data and Building a Dashboard
Project Overview

This project analyzes historical stock price data and company revenue data to better understand how market performance relates to company financial performance. Using Python and data analysis libraries, the notebook extracts, processes, and visualizes stock and revenue data, and then builds an interactive dashboard to display the trends.

The goal is to demonstrate how financial data can be collected from online sources, cleaned, analyzed, and visualized to provide insights into company performance.
